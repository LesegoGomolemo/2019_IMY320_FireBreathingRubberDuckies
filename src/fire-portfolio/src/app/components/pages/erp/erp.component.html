Fire Portfolia | ERP

Project Goal:


Our project is to create a mobile application to assist ERP rangers with various daily tasks. These include, managing a ranger's patrol shifts, tracking a ranger's position whilst out on patrol and allowing rangers to write reports after a shift has ended. It is also an effort to gameify the patrolling of the park as all rangers using the application are volunteers, the gamification comes in the form of points that can be earned by the rangers while on patrol.

The Project:


This project Involved the development of a Backend that would be used by Administrators. They would be able to monitor people currently in the park as well as any incidents reported by rangers that are out on patrol. It also required the development of a mobile application that would be used by rangers on patrol, they are able to report incidents, book shifts ahead of time, receive alerts and earn points by activating markers placed in the park by administrators. These points can be redeemed on the ERP merchandise store for material rewards.

Project Page:


Currently the project is based on github with directions to contribute and use the project. Because the dart framework was used the project is compatible with both Android and IOS systems.
