Project Specification:


We were tasked to design this website with a few specific projects in mind which we were expected to already have completed as either multimedia or CS students, however, because none of the team studied multimedia, none of us had a game to implement or show here. Thus we decided to appropriate this flash game and pass it off as our own purely for credit in our IMY320 module.

The Project:


The goal of the game is to align the "bubbles" into groups of 3 or more in the smallest amount of time. The player gets to "Launch" bubbles towards those already on screen and when 3 or more "bubbles" of the same color are connected in this manner the "pop" and are removed from the board, the player wins the level when all "bubbles" are removed from the boards. The player loses however if the entire board is filled with bubbles and reaches the marked area at the bottom of the board.

Project Page:


The project is hosted on html games, and cannot be contributed to further. However it is still playable and is very enjoyable.
